This is intended for Ubuntu (.deb) based systems. This was initially built in Linux Mint 22

1. Run the appropreate script for your system:

   AMD GPU: amd-install.sh
   NVIDIA GPU: nvidia-install.sh

2. Make it executatble by right clicking "Properties" and then checking "Allow executing file as program"

3. Double Click to Install

   This will install appropreate dependicies and DaVinci Resolve
   You may have to enter "y" for yes when prompted

That's it, find DaVinci Resolve under "Sound & Video" in Menu
